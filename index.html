Fixture
A no bullshit raster graphics editor.
Fixture is a cross-platform free and open source raster graphics editor. It is inspired by the need for having an equivalent of Adobe Photoshop on Linux-based platforms.
Fixture aims to be solid enough for season veterans , quick enough for amateur graphic artists and easy for new users to learn. It can be ported to any of your favorite operating systems that supports Qt. And it's free!

The project is hosted at github and is based on Qt 5.11 on C++.
